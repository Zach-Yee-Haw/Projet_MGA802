extra_fonctions module¶

extra_fonctions.choix_biaises(liste_de_choix, nb_de_choix, biais)¶

Sélectionne un certain nombre d’éléments d’une liste avec un biais donné.

Paramètres:

  • liste_de_choix (list) – Liste des éléments parmi lesquels choisir.

  • nb_de_choix (int) – Nombre d’éléments à sélectionner.

  • biais (float) – Facteur de biais pour influencer les choix (plus bas, plus les premiers éléments sont favorisés).

Renvoie:

Liste des éléments sélectionnés avec biais.

Type renvoyé:

list

extra_fonctions.melanger_structures(structures, nb_structures)¶

Mélange les structures de manière aléatoire.

  • nb_structures (int) – Nombre de structures à mélanger.

Renvoie:

Nouveau tableau contenant les structures mélangées.

Type renvoyé:

ndarray
